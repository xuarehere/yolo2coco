# 实现
## STEP 1 切分数据集
生成训练集、验证集、测试集

将 split_dataset.py 拷贝到 `./data`
在该工程下运行
```
python split_dataset.py
```

生成如下的文件结果
```
output:
./data
├── images
├── labels
├── test
│   ├── images
│      └───*.jpeg
│   └── labels
│   |  └───*.txt
│   └── label.txt
├── train
│   ├── images
│      └───*.jpeg
│   └── labels
│   |   └───*.txt
│   └── label.txt
└── val
    ├── images
    |   └───*.jpeg    
    └── labels
    |  └───*.txt 
    └── label.txt         
```





## STEP 2  生成 coco 格式

```
python yolo2coco.py
```

